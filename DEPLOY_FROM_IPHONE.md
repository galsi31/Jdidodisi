# Размещение с iPhone без компьютера

## Вариант 1 — GitHub Pages

1. В приложении «Файлы» распакуйте ZIP `100K_RPG_App.zip`.
2. Откройте github.com в Safari и войдите в аккаунт.
3. Создайте новый публичный репозиторий, например `100k-rpg`.
4. Нажмите **Add file → Upload files**.
5. Загрузите содержимое распакованной папки:
   - index.html
   - styles.css
   - app.js
   - manifest.webmanifest
   - service-worker.js
   - icon-192.png
   - icon-512.png
   - .nojekyll
6. Откройте **Settings → Pages**.
7. В разделе Build and deployment выберите:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
8. Сохраните. Через некоторое время появится ссылка вида:
   `https://ВАШ-ЛОГИН.github.io/100k-rpg/`

## Установка на iPhone

1. Откройте опубликованную ссылку именно в Safari.
2. Нажмите значок «Поделиться».
3. Выберите «На экран Домой».
4. Включите «Открывать как веб‑приложение».
5. Нажмите «Добавить».

## Резервная копия

В приложении откройте:
**Профиль → Сохранение → Экспорт**.

Перед очисткой Safari, сменой телефона или удалением приложения обязательно экспортируйте JSON-файл.
