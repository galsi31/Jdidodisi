# 100K RPG

Личное мобильное PWA-приложение для финансов, продуктивности, целей внешности и дисциплины.

## Что уже работает

- повторяющийся денежный босс на каждые 7 дней;
- редактируемые цели и привычки;
- XP, уровни, ранги, комбо, монеты, достижения;
- доходы, расходы, фонд переезда;
- ежедневные задачи, калории и тренировки;
- журнал квестов со сложностью и XP;
- история всех семидневных сезонов;
- магазин личных наград;
- локальное сохранение;
- экспорт и импорт резервной копии;
- необязательный PIN;
- офлайн-режим после первого запуска;
- установка на главный экран iPhone.

## Важное ограничение приватности

Личные данные не входят в исходный код и не загружаются на GitHub. Они сохраняются в браузере конкретного устройства.

PIN — это блокировка интерфейса приложения, а не полноценное шифрование хранилища. Защита телефона Face ID и кодом остаётся основной защитой.

## Быстрый запуск

Откройте `index.html` через локальный сервер или опубликуйте содержимое папки через GitHub Pages / Cloudflare Pages.

Подробная инструкция с телефона находится в `DEPLOY_FROM_IPHONE.md`.
